0x16. C - Simple Shell Program

Project Description: This project seeks to create a simple shell program that takes in input from the command-line and connects it directly with the operating system to produce an output. In this instance, this project will re-create the simple shell version of the linux shell using the C programming language.

Task 0. Betty

        Writing a code according to the specifications found in Betty documentation

Task 1. Command-line interpreter

        Creating a UNIX command-line interpreter

Task 2. Command-line and arguments

        Working with command lines with arguments

Task 3. PATH

        Working with the PATH

Taks 4. Exit function

        Creating the exit built-in function to exit the shell

Task 5. Show current environment

        Creating the env built-in function to display the current environment
